### 通用错误代码

| 代码 | 说明 |
| --- | ---- |
| 0 | 请求成功 |
| 500 | 服务器错误 |
| 1000 | token验证失败 |
| 1001 | 无权限 |
| 1002 | 请求资源不存在 |
| 1003 | 请求次数超限 |
| 1004 | 数据已被更新 |
| 1101 | 排序值必须为一个整数 |
| 1102 | 排序值必须是 0（含） - 255（含） 之间的整数 |
| 1103 | 备注最大只能 255 个字符 |

### 登录

| 代码 | 说明 |
| --- | --- |
| 10100 | 用户名不能为空 |
| 10101 | 用户名不能小于4个字符 |
| 10102 | 用户名不能超过45个字符 |
| 10103 | 密码不能为空 |
| 10104 | 密码不能超过255个字符 |
| 10105 | 登录设备不能为空 |
| 10106 | 登录设备不能超过100个字符 |
| 10107 | 用户不存在或禁止登录 |
| 10108 | 登录失败 |
| 10109 | 需要修改密码 |
